# Bomberman

Simple Bomberman implementation using Godot game engine

![Preview](/doc/preview.png)

# Controls

* ZQSD or arrow keys to move
* Space to spread chaos and destruction

## Feature

* Player animations
* Enemy animations
* Wall
* Level generation
* Bomb
* Limitation on bomb placement
* Power-UPS
* Sickness
* Kick

## Planned features
